= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress(This will add new Sub Menu in admin panel's side menu name Rebrand` under Salon menu).
3. Go to Salon > Rebrand
